# TPA
A TPA Plugin for PocketMine

# Commands
|Command|Description|Permission|
|-----------|---------------|----------|
|``/tpa``|send a Player a tpa|tpa.tpa.use|
|``/tpahere``|send a Player a tpa here|tpa.tpahere.use|
|``/tpaccept``|accept a tpa|tpa.tpaccept.use|
|``/tpadeny``|deny a tpa|tpa.tpadeny.use|
